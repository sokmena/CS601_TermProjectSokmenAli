Student Data Entry Form

This is a form validation exercise I have done for the Introduction to Web Application Development class I have taken at BU. Using Javascript DOM manipulation and Regex, I check for the validity of form content and provide feedback to the user. If validated, the data is sent to server and the user is provided with feedback (outside application).

Fetch Academic Data

Another simple application done for the BU Web Development class, this time focusing on fetching data using APIs and promises. The application fetches a JSON file (in a web address but hosted in the same application), and writes the data in it into an HTML table. The size of the table, both in columns and rows, can undergo changes based on to the data structure and number of entries.
